Instalar Minimal Kiosk Browser en Raspbian

2014-09-19 Utilidades Raspberry Pi

25 lineas 99 palabras 747 letras

Problema: Queremos ver webs y alguna cosilla más desde nuestra Raspbery pero no queremos arrancar un entorno de escritorio completo

Solución: Instalar Minimal Kiosk Browser

pi@bot ~ $ cd /tmp
pi@bot /tmp wget http://steinerdatenbank.de/software/kweb-1.5.4.1.tar.gz
...
pi@bot /tmp tar -xzf kweb-1.5.4.1.tar.gz
pi@bot /tmp/kweb-1.5.4.1 ./prepare
Nos dirá que paquetes nos falta
pi@bot /tmp/kweb-1.5.4.1 sudo apt-get install paquetes que nos faltan
...
pi@bot /tmp/kweb-1.5.4.1 sudo ./install

Para ejecutar sin tener que arrancar el entorno de escritorio:

pi@bot ~ $ xinit ./ktop

O para facilitar su uso podemos crear un alias

~/.bashrc

alias ktop="xinit ~/ktop"

Documentación: kweb manual, omxplayerGUI manual, changelog

Fuente: www.raspberrypi.org
