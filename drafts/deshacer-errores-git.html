Chiquirecopilatorio de comandos útiles para cuando metes la pata usando Git

Deshacer un push

Listamos los dos últimos commit:

$ git rebase -i HEAD~2

Seleccionamos el segundo y lo borramos.

Forzamos el cambio en la rama master o aquella en la que queramos hacer la corrección:

git push origin +master --force
