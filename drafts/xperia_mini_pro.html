Pasos a seguir

Prerrequisitos

# Formatear SD con una unica partición en formato ext4
$ sudo umount /dev/mmcblk0p1
$ sudo mkfs -t ext4 /dev/mmcblk0
$ sudo umount /dev/mmcblk0

# Descomprimir lineage-*.zip
$ unzip lineage-14.1-20170514-UNOFFICIAL-LegacyXperia-mango.zip
$ cd lineage-14.1-20170514-UNOFFICIAL-LegacyXperia-mango

# Instalar adb y fastboot
$ sudo apt install adb fastboot

Verificar dispositivo

  1. Encender el movil
  2. Conectar al ordenador por usb
  3. Marcar Ajustes -> opciones de desarrollador -> Depuración USB
  4. Ejecutar los siguientes comandos:
$ adb devices
List of devices attached
BX902QUGBW  device
$ adb reboot bootloader
$ fastboot devices

Si adb reboot bootloader no funciona (no reinicia en modo bootloader, simplenete reinicia sin más) verificar que el modo bootloader este desbloqueado:

  1. Marcar *#*#7378423#*#*
  2. Ir a Service info" -> Configuration -> Rooting Status -> Bootloader unlock allowed

Si el valor es Yes esta desbloqueado, si es No hacer lo siguiente:

  1. Apagar el movil
  2. Simultaneamente:
    • presional la tecla de aumentar volumne
    • enchufar por usb al ordenador

powered off, plugging in USB cable while pressing volume-up, blue LED will indicate fastboot mode

$ unzip lineage-14.1-20170514-UNOFFICIAL-LegacyXperia-mango.zip

# Flash boot.img
$ fastboot flash boot boot.img

  1. External memory card is required to install the rom (Ext4 only 1 partition). At least around 4gb, must be very fast (Class 10 u3), and should be at around 8gb if possible. All apps, data, etc will go here.

  2. Flash boot.img (In fastboot mode).

  3. Go wipe, swipe. Go advanced wipe and wipe system.

  4. Go advanced --> sideload. Connect USB and type 'adb sideload rom-14.1-xxx.zip' & after type 'adb sideload opengapps-7.1-xxx.zip'

  5. Reboot to system. Ignore any warning message.
